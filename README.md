# 📝 CLI Task Manager

A lightweight, interactive Command-Line Interface (CLI) Task Manager built with Python. This application allows users to manage their daily tasks efficiently directly from the terminal, featuring robust input validation and an intuitive user experience.

---

## 🚀 Features

*   **Add Tasks:** Quickly add new tasks with empty-input validation.
*   **View Tasks:** Display your current task list with clean, numbered formatting.
*   **Delete Tasks:** Remove completed or unwanted tasks by selecting their corresponding list number.
*   **Error Handling:** Built-in protection against invalid menu choices, empty entries, and non-numerical inputs.
